This directory contains some  documents  and  resources  that  can  be  used  to
understand how the MQTT protocol works.

 file                 |  summary
----------------------|---------------------------------------------------------
 MQTT_3.1.1.mhtml     | bundled html file of MQTT 3.1.1 RFC
 urls.txt             | list of urls of external resources on MQTT
 mqtt_capture.pcapng  | Wireshark capture of MQTT traffic
